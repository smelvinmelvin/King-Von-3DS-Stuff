# King-Von-3DS-Stuff
King Von theme and splash for the 3DS

## Installation
(Requires a [modded 3ds](https://3ds.hacks.guide/)!!)
Move the theme zip to sdmc:/Themes
Move the splash zip to sdmc:/Splashes
Then go to Anemone and install there.
