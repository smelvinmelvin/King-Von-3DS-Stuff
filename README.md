# King Von 3DS stuff
King Von theme and splash for the 3DS

## Installation
(Requires a [modded 3ds](https://3ds.hacks.guide/)!!)
1. Move the theme zip to sdmc:/Themes
2. Move the splash zip to sdmc:/Splashes
3. Then go to Anemone and install them there.
